## Model Overview
The Western Digital WD16 was an LSI-11 compatible processor that offered DEC PDP-11 instruction set compatibility in a single chip. It provided a cost-effective way to build PDP-11 compatible systems using a microprocessor instead of a multi-chip minicomputer design, targeting embedded and industrial applications.
