## Model Overview
The Hitachi HD64180 was a Z180-compatible enhanced Z80 microprocessor with an 8-bit data bus and 20-bit address bus (1 MB addressable via on-chip MMU). It offered faster instruction execution than the Z80, with on-chip MMU, DMA, UART, and timers. It was used in embedded systems, industrial controllers, and Japanese computers.
