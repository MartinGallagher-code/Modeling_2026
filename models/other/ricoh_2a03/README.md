## Model Overview
The Ricoh 2A03 was an 8-bit microprocessor based on the MOS 6502 core with decimal mode (BCD) disabled. It ran at 1.79 MHz (NTSC) or 1.66 MHz (PAL) and included an on-chip Audio Processing Unit (APU) for sound generation. The instruction timing was identical to the MOS 6502. It is best known as the CPU of the Nintendo Entertainment System (NES) and Famicom.
