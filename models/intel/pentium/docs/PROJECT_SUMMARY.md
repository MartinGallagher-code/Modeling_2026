# Intel Pentium - Project Summary

## Overview
First superscalar x86 processor (1993). Defined 1990s PC computing.

## Key Specifications
- **Transistors:** 3,100,000
- **Pipelines:** 2 (U and V)
- **IPC:** >1.0 (superscalar)
- **Clock:** 60-200 MHz
- **Data Bus:** 64-bit

## Key Innovations
1. Dual execution pipelines
2. Dynamic branch prediction
3. Separate I/D caches
4. Pipelined FPU
