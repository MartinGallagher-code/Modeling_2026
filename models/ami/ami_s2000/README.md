# AMI S2000

4-bit calculator chip (1971) - one of the earliest calculator ICs.

## Overview

| Property | Value |
|----------|-------|
| Manufacturer | American Microsystems Inc. |
| Year | 1971 |
| Data Width | 4-bit |
| Clock | 200 kHz |
| Target CPI | 8.0 |
| Technology | PMOS |

## Description

The AMI S2000 was one of the earliest 4-bit calculator chips, predating the Intel 4004. Built on PMOS technology with a 200 kHz clock, it was very slow by later standards but was a pioneering device for electronic calculators.
