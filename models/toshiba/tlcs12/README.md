# Toshiba TLCS-12

## Overview
The Toshiba TLCS-12 (1973) was the first Japanese microprocessor. A 12-bit PMOS device with approximately 2500 transistors, it was designed primarily for the Ford EEC (Electronic Engine Control) system for automotive engine management.

## Specifications
- **Manufacturer**: Toshiba
- **Year**: 1973
- **Data Width**: 12-bit
- **Clock Speed**: 1 MHz
- **Technology**: PMOS
- **Transistors**: ~2,500

## Model Details
- **Target CPI**: 8.0
- **Architecture**: Sequential execution, no pipeline
- **Workloads**: typical, compute, memory, control
