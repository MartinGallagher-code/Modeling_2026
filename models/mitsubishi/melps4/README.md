# Mitsubishi MELPS 4 (M58840)

4-bit microcontroller (1978) - Mitsubishi's first 4-bit MCU, PMOS technology.

| Property | Value |

| Technology | PMOS |

## Description

The MELPS 4 (M58840) was Mitsubishi's first 4-bit microcontroller. Built on PMOS technology, it was relatively slow but adequate for consumer electronics and appliance control. Variable instruction timing ranges from 4-8 cycles depending on instruction type.

## Instruction Categories

| Category | Cycles |
|----------|--------|
| alu | 4 |
| data_transfer | 5 |
| memory | 7 |
| io | 8 |
| control | 6 |
