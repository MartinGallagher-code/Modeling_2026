# Mitsubishi MELPS 740 (M50740)

## Overview
The Mitsubishi MELPS 740 / M50740 (1984) was an enhanced 6502-compatible 8-bit microcontroller built in CMOS technology. It extended the MOS 6502 instruction set with MUL, DIV, and bit manipulation instructions, and included on-chip peripherals such as timers, serial I/O, and A/D converter. It was widely used in consumer electronics and appliance control applications.

## Specifications
- **Manufacturer**: Mitsubishi Electric
