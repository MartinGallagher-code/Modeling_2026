# Mitsubishi MELPS 42

4-bit microcontroller (1983) - CMOS version of MELPS 4 family.

| Property | Value |
|----------|-------|

| Technology | CMOS |

## Description

The MELPS 42 is the CMOS version of Mitsubishi's MELPS 4 family. It offers improved clock speed (1 MHz), lower power consumption, and slightly better instruction timing (CPI 5.0) compared to the NMOS MELPS 41.
