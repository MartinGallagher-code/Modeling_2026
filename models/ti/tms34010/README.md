# TI TMS34010

## Overview
- **Type**: 32-bit Graphics Processor (First Programmable GPU)
- **Year**: 1986
- **Manufacturer**: Texas Instruments
- **Clock**: 6 MHz
- **Transistors**: ~275,000

## Architecture
First programmable graphics processor. 32-bit general-purpose CPU with hardware pixel operations, bit-level addressing, and graphics pipeline. Used in arcade games (Mortal Kombat, NBA Jam) and PC graphics cards.
