# Motorola 6801 - Quick Start

## First MCU with On-Chip RAM (1978)

True single-chip computer: CPU + ROM + RAM + UART + Timer

---

## Quick Facts

| Spec | Value |
|------|-------|
| Year | 1978 |
| **On-chip RAM** | **128 bytes (FIRST!)** |
| On-chip ROM | 2 KB |
| Clock | 1-2 MHz |
| New instruction | **MUL** (10 cycles) |

---

## vs 6800

| Feature | 6800 | 6801 |
|---------|------|------|
| On-chip RAM | ✗ | **✓ 128B** |
| Hardware MUL | ✗ | **✓** |
| On-chip UART | ✗ | **✓** |
| Single-chip | ✗ | **✓** |
