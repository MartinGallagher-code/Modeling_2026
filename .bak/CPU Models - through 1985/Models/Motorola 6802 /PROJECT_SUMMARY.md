# Motorola 6802 - Project Summary

## 6800 with Integrated RAM and Clock (1977)

Simplified 6800 system design by adding 128 bytes RAM and on-chip clock generator.

## Key Features
- 128 bytes on-chip RAM
- On-chip clock generator
- Single +5V power supply
- RAM standby mode

## vs 6801
- 6802 = CPU (RAM only)
- 6801 = MCU (RAM + ROM + I/O)
