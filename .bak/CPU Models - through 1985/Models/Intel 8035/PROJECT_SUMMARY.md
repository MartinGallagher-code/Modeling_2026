# Intel 8035 - Project Summary

ROM-less version of Intel 8048 (1976).

## Key Features
- No internal ROM (external required)
- 64 bytes internal RAM
- Same peripherals as 8048
- MCS-48 instruction set

## Part of MCS-48 Family
8048 (ROM) / 8035 (ROM-less) / 8748 (EPROM)
8049 (enhanced) / 8039 (enhanced ROM-less)
