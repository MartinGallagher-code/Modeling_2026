# Intel 8096 - Quick Start

## 16-bit Automotive MCU (1982)

Dominated engine control for 20+ years!

| Feature | Automotive Use |
|---------|----------------|
| A/D | Sensors |
| HSI | Crankshaft position |
| HSO | Spark timing |
| PWM | Idle control |
| Multiply | Fuel calculations |

## Specs
- 16-bit CPU
- 8 KB ROM, 232 bytes RAM
- 10-bit A/D (8 channels)
- Hardware multiply/divide

## Why It Won
16-bit math + fast A/D + precise timing = perfect for engines.
