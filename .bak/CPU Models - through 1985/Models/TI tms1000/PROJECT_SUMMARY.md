# TI TMS1000 - Project Summary

## First Mass-Produced Microcontroller (1974)

BILLIONS shipped. Made embedded computing ubiquitous.

## Complete Single-Chip Computer
- 4-bit CPU
- 1KB ROM
- 64 nibbles RAM
- I/O ports
- Clock oscillator
- ALL FOR UNDER $2!

## Impact
Powered Speak & Spell, Simon, TI calculators, and countless appliances. Made computers truly everywhere.
