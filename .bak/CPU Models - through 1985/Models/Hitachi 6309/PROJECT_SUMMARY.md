# Hitachi 6309 - Project Summary

## Best 8-Bit Processor Ever Made (1982)

The 6309 was sold as a 6809 second-source but secretly contained massive enhancements, undocumented until 1988.

## Secret Features
- **6 new registers** (E, F, W, Q, V, MD)
- **32-bit Q register** (unprecedented in 8-bit)
- **Hardware divide** (6809 had none)
- **TFM block transfer** (3 cycles/byte)
- **~60 new instructions**

## The Mystery
Hitachi kept features secret to avoid upsetting Motorola licensing. Users discovered them by accident in 1986.
