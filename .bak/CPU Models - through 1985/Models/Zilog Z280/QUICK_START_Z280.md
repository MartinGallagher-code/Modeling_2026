# Zilog Z280 - Quick Start

## Failed Z80 Successor (1985)

| Feature | Z80 | Z280 |
|---------|-----|------|
| Bus | 8-bit | **16-bit** |
| Address | 64 KB | **16 MB** |
| MMU | None | **On-chip** |
| Cache | None | **256B** |

**Why it failed:**
- Not fully Z80 compatible
- 1985 market wanted 80286/68000
- Too little, too late
