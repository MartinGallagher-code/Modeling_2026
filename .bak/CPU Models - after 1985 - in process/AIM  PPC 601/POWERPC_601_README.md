# PowerPC 601 CPU Queueing Model

## Executive Summary
The PowerPC 601 (1993) was the first processor from the **Apple-IBM-Motorola (AIM) alliance**. Three-way superscalar execution made it faster than the Pentium in many benchmarks. It powered the first Power Macintosh computers, ending the 68k era.

## Technical Specifications
| Spec | Value |
|------|-------|
| Year | 1993 |
| Transistors | 2,800,000 |
| Clock | 66-120 MHz |
| Issue width | **3** |
| Cache | 32 KB unified |
| Data bus | 64-bit |

## Three-Way Superscalar
```
Can execute simultaneously:
- Integer operation
- Floating-point operation  
- Branch operation

Up to 3 instructions per cycle!
```

## The AIM Alliance
```
1991: Apple, IBM, Motorola form alliance
1992: PowerPC architecture defined
1993: 601 ships
1994: Power Macintosh launches

Goal: Compete with Intel
Result: Successful for 10+ years
End: Apple switches to Intel (2006)
```

## Power Macintosh (1994)
- Power Mac 6100: 60 MHz
- Power Mac 7100: 66 MHz
- Power Mac 8100: 80 MHz

First Macs faster than equivalent PCs!
