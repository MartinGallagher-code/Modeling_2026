# Mitsubishi MELPS 41

4-bit microcontroller (1980) - Enhanced MELPS 4, NMOS technology.

## Overview

| Property | Value |
|----------|-------|
| Manufacturer | Mitsubishi |
| Year | 1980 |
| Data Width | 4-bit |
| Clock | 500 kHz |
| Target CPI | 5.5 |
| Technology | NMOS |

## Description

The MELPS 41 is an enhanced version of the MELPS 4, upgraded from PMOS to NMOS technology. This provides improved speed (500 kHz vs 400 kHz) and slightly better instruction timing (CPI 5.5 vs 6.0).
