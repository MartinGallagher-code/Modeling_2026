# AMI S2150

4-bit calculator chip (1972) - S2000 variant.

## Overview

| Property | Value |
|----------|-------|
| Manufacturer | American Microsystems Inc. |
| Year | 1972 |
| Data Width | 4-bit |
| Clock | 200 kHz |
| Target CPI | 8.0 |

## Description

The AMI S2150 is a variant of the S2000 calculator chip with minor enhancements. It shares the same instruction timing (CPI 8.0) and PMOS technology as the base S2000.
