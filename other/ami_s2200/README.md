# AMI S2200

4-bit calculator chip (1972) - S2000 variant with more ROM.

## Overview

| Property | Value |
|----------|-------|
| Manufacturer | American Microsystems Inc. |
| Year | 1972 |
| Data Width | 4-bit |
| Clock | 200 kHz |
| Target CPI | 8.0 |
| Technology | PMOS |

## Description

The AMI S2200 is an S2000 variant with expanded ROM capacity. It shares the same instruction timing (CPI 8.0) and PMOS technology as the base S2000.
