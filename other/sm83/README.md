# Sharp SM83 (LR35902)

## Overview
- **Type**: 8-bit processor (Game Boy CPU)
- **Year**: 1989
- **Manufacturer**: Sharp
- **Clock**: 4.194304 MHz
- **Transistors**: ~8,000

## Architecture
Custom Sharp CPU for the Nintendo Game Boy. Hybrid Z80/8080 ISA - has Z80-like instructions but missing IX/IY registers and Z80 I/O. Adds unique SWAP instruction. Used in Game Boy, Game Boy Color, and Super Game Boy.
