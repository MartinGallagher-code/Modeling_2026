# HP PA-RISC CPU Queueing Model

## Executive Summary
HP PA-RISC (Precision Architecture RISC, 1986) was Hewlett-Packard's RISC processor family that powered HP workstations and servers for over two decades. Known for excellent floating-point performance and tight integration with HP-UX.
