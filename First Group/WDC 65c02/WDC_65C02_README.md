# WDC 65C02 CPU Queueing Model

## Executive Summary
The 65C02 (1983) was the **CMOS version of the legendary 6502**, designed by Bill Mensch at Western Design Center. It added new instructions, fixed bugs, and enabled low-power applications like the Apple IIc.

## Technical Specifications
| Spec | Value |
|------|-------|
| Year | 1983 |
| Technology | **CMOS** (vs NMOS) |
| Clock | Up to 14 MHz |
| Power | Much lower than 6502 |
| Static | Can halt clock |

## Key Improvements Over 6502
- **CMOS**: Low power, static operation
- **New instructions**: PHX, PHY, PLX, PLY, STZ, BRA
- **Bug fixes**: JMP indirect crossing page boundary
- **Higher clocks**: 14 MHz vs 2-3 MHz
- **Rockwell extensions**: Bit manipulation (BBR, BBS, RMB, SMB)

## Famous Systems
- **Apple IIc** (portable Apple II)
- **Apple IIe Enhanced**
- **Commander X16** (modern retro)
- Many embedded systems

## Still Produced!
WDC still manufactures the 65C02 in 2026. The 6502 lives on!
