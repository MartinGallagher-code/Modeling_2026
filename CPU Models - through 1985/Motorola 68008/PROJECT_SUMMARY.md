# Motorola 68008 - Project Summary

## 8-Bit Bus 68000 (1982)

The 68008 is the cost-reduced version of the 68000, using an 8-bit external bus instead of 16-bit.

## Key Differences from 68000
- 8-bit data bus (vs 16-bit)
- 1 MB address space (vs 16 MB)
- 48-pin package (vs 64-pin)
- ~40% of 68000 performance

## Advantages
- Lower chip cost
- Simpler PCB design
- 8-bit memory compatibility
- Same software as 68000
