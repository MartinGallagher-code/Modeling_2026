# AMD Am2901 - Quick Start

## 4-Bit ALU Slice (1975)

Build your own CPU by cascading slices!

| Slices | Width | Example Use |
|--------|-------|-------------|
| 4 | 16-bit | Workstations |
| 8 | 32-bit | Minicomputers |
| 16 | 64-bit | Supercomputers |

## Inside Each Slice
- 16 x 4-bit registers
- 4-bit ALU (8 functions)
- Shifter
- Q register

## Why Bit-Slice?
- Fastest technology (Schottky TTL)
- Custom instruction sets
- Used in VAX, Cray, arcade games
