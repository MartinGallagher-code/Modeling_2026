# GI CP1600 - Quick Start

## Intellivision CPU (1975)

Early 16-bit processor that powered 3+ million game consoles.

| Spec | Value |
|------|-------|
| Bits | 16 |
| Clock | 895 kHz |
| Registers | 8 × 16-bit |
| Famous for | Intellivision |

## Legacy
- Intellivision sold 3M+ units
- Needed I/O controller → GI created PIC1650
- PIC became more successful than CP1600!
