# RCA CDP1806 - Project Summary

## Final COSMAC Family Member (1985)

Ultimate evolution of the space-proven COSMAC family.

## Enhancements Over 1802
- 128 bytes on-chip RAM (vs none)
- 91 instructions (vs 46)
- Enhanced timer
- 8 MHz operation (vs 2 MHz)

## Space Heritage
Voyager probes still running after 45+ years. New Horizons is 8+ billion km away. Silicon-on-Sapphire provides radiation immunity.

## Still Available
Radiation-hardened versions still manufactured for space/military.
