# NEC V20 - Quick Start

## Drop-In 8088 Replacement (1984)

10-40% faster, hardware 8080 mode, lawsuit survivor!

---

## Quick Specs

| Spec | 8088 | V20 |
|------|------|-----|
| MUL 16×16 | 118 cyc | **25 cyc (4.7×)** |
| DIV 32/16 | 150 cyc | **30 cyc (5×)** |
| Shifts | 4+n cyc | **1 cycle** |
| 8080 mode | No | **Yes** |

---

## Key Features

- **Pin compatible** with 8088
- **4-5× faster** multiply/divide  
- **Hardware 8080** emulation
- **New instructions** (bit ops, block I/O)
