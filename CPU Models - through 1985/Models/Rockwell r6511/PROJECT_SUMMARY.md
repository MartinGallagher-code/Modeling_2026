# Rockwell R6511 - Project Summary

## 6502 MCU Variant (1980)

Rockwell's embedded-focused 6502 variant with integrated peripherals.

## On-Chip Integration
- 192 bytes RAM
- 32 I/O lines
- Full UART
- Two 16-bit timers

## Significance
Reduced 6502 embedded systems from 5+ chips to 1, enabling cost-effective industrial applications.
