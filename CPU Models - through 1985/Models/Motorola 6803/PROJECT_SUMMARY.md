# Motorola 6803 - Project Summary

ROM-less version of 6801 (1979).

## Key Differences from 6801
- No on-chip ROM (external required)
- Fewer I/O pins (13 vs 29)
- Same 128B RAM and peripherals

## Use Cases
- Programs larger than 2KB
- Development systems
- Flexible ROM requirements
