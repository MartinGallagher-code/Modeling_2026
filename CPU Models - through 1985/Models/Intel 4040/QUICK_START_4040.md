# Intel 4040 - Quick Start

## Enhanced 4004 (1974)

| Feature | 4004 | 4040 |
|---------|------|------|
| Stack | 3 | **7 levels** |
| Interrupts | No | **Yes** |
| Instructions | 46 | **60** |

Key improvement: **Interrupts** enabled real embedded applications.

New: HLT, EIN, DIN, BBS, WPM, RPM instructions.
