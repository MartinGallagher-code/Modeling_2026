# NSC NSC800 - Quick Start

## Z80-Compatible, CMOS (1979)

| Spec | Z80 | NSC800 |
|------|-----|--------|
| Software | Z80 | **Compatible** |
| Process | NMOS | **CMOS** |
| Pinout | Z80 | **8080-like** |
| Power | Higher | **Lower** |

## Key Differences
- NOT pin-compatible (8080 pinout)
- Can stop clock (static design)
- Single-phase clock (simpler)

## Use Cases
Battery devices, military, industrial.
