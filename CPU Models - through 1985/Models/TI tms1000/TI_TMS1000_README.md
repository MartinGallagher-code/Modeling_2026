# Texas Instruments TMS1000 Queueing Model

## THE FIRST MASS-PRODUCED MICROCONTROLLER (1974)

The TMS1000 was the first true single-chip microcontroller, and **BILLIONS** have been shipped. It made embedded computing ubiquitous.
