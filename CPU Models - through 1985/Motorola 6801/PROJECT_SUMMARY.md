# Motorola 6801 - Project Summary

## First MCU with On-Chip RAM (1978)

The 6801 was the first microcontroller with integrated RAM, enabling true single-chip computers.

## Key Innovations
- **128 bytes on-chip RAM** (industry first)
- Hardware 8Ã—8 multiply (MUL instruction)
- On-chip full-duplex UART
- 16-bit timer
- 6800 instruction superset

## Historical Impact
- Enabled low-cost embedded systems
- Foundation for 68HC11 family
- Established Motorola in automotive

## Variants
- 6801: 2KB mask ROM
- 68701: EPROM version
- 6803: ROM-less
