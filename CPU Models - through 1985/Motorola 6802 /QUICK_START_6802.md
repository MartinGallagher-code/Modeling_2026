# Motorola 6802 - Quick Start

## 6800 + RAM + Clock (1977)

| Feature | 6800 | 6802 |
|---------|------|------|
| RAM | None | **128B** |
| Clock | External | **On-chip** |
| Power | 3 rails | **+5V only** |

100% 6800 software compatible.
Simplified system design.
Standby mode for RAM retention.

**6802 = CPU, 6801 = MCU (has I/O)**
