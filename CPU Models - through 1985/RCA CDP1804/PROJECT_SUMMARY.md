# RCA CDP1804 - Project Summary

## Enhanced Space Processor (1980)

Middle member of COSMAC family, between 1802 and 1805.

## Enhancements Over 1802
- 64 bytes on-chip RAM
- On-chip timer
- Higher clock (5 MHz)
- Optional mask ROM

## Radiation Hardened
Silicon-on-Sapphire (SOS) CMOS provides natural radiation immunity without special processing.

## Space Heritage
The 1802 family powers spacecraft from Voyager to New Horizons. The 1804 brought improvements for newer missions.
