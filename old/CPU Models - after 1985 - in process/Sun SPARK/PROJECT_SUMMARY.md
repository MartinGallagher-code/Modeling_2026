# SPARC - Project Summary

## Overview
Sun's open RISC architecture (1987). Defined Unix workstation era.

## Key Innovations
- **Register windows:** 136 registers, procedure call without memory access
- **Open architecture:** First freely licensable processor
- **Delayed branches:** Better pipeline utilization

## Impact
- Dominated Unix workstations
- Multiple vendors (Sun, Fujitsu, TI)
- Influenced open hardware (RISC-V)
