# GI PIC1650 - Quick Start

## First PIC Microcontroller (1977)

Ancestor of billions of MCUs!

| Spec | Value |
|------|-------|
| Instructions | 33 |
| Program Mem | 512 words |
| Data RAM | 32 bytes |
| Architecture | Harvard |

## Why It Won
- Simple (33 instructions)
- Cheap (pennies)
- Available (everywhere)
- Great docs

## Evolution
PIC1650 → PIC16 → PIC18 → PIC32
**Billions shipped!**
