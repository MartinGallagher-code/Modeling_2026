# WDC 65816 CPU Queueing Model

## Executive Summary
The 65816 (1984) was the **16-bit evolution of the 6502**, designed by Bill Mensch. It powered the Apple IIGS and, in a modified form (Ricoh 5A22), the **Super Nintendo**.

## Technical Specifications
| Spec | Value |
|------|-------|
| Year | 1984 |
| Word Size | **16-bit** |
| Data Bus | 8-bit |
| Address | **24-bit (16MB)** |
| Compatibility | Full 6502 emulation |

## Key Features
- **16-bit registers** (switchable 8/16)
- **16MB address space** (vs 64KB)
- **Direct page relocatable** (enhanced zero page)
- **Full 6502 compatibility** (emulation mode)

## Famous Systems
- **Apple IIGS** (1986) - Last Apple II
- **Super Nintendo** (1990) - Ricoh 5A22 variant
- Various accelerator cards

## The SNES Connection
The Super Nintendo used a Ricoh 5A22, which was a 65816 with:
- Custom DMA
- Multiplication/division hardware
- 3.58 MHz clock

The SNES sold 49 million units!

## Still Produced
WDC still manufactures the 65816 in 2026.
