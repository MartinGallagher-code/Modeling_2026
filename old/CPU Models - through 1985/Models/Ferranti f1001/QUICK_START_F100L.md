# Ferranti F100-L - Quick Start

## British Military Processor (1976)

| Spec | Value |
|------|-------|
| Bits | 16 |
| Clock | 1.6 MHz |
| Rad-hard | Yes |
| Origin | UK |

## Applications
- Tornado aircraft
- Rapier missile
- UK naval systems
- Satellites

## Why British?
Cold War policy: UK military needed processors not dependent on foreign supply.

Reliability > Speed
