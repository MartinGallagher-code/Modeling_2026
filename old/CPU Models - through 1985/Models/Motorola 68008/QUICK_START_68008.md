# Motorola 68008 - Quick Start

8-bit bus version of 68000 (1982).

| Spec | 68000 | 68008 |
|------|-------|-------|
| Internal | 32-bit | 32-bit |
| Bus | 16-bit | **8-bit** |
| Address | 16 MB | **1 MB** |
| Speed | 1.0× | **~0.4×** |
| Cost | Higher | **Lower** |

Same instructions, same registers, smaller bus.
Famous system: Sinclair QL (1984).

**68008 : 68000 :: 8088 : 8086**
