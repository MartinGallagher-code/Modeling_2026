# GI CP1600 - Project Summary

## Intellivision CPU (1975)

Early 16-bit microprocessor, most famous as the CPU in Mattel's Intellivision console.

## Key Facts
- True 16-bit architecture
- 8 general-purpose registers
- Powered 3+ million Intellivisions
- Spawned the PIC microcontroller family

## Historical Note
The PIC1650, designed as an I/O controller for the CP1600, went on to become far more successful - billions shipped!
