# Motorola 68030 - Project Summary

## Overview
68020 with on-chip MMU and data cache (1987). The "classic" Mac CPU.

## Key Specifications
- **Transistors:** 273,000
- **Clock:** 16-50 MHz
- **Caches:** 256B I-cache + 256B D-cache
- **MMU:** On-chip (vs external 68851)
- **IPC:** ~0.80

## Key Improvements Over 68020
1. On-chip MMU (simpler, cheaper systems)
2. Data cache (faster memory access)
3. Burst mode (faster cache fills)
4. Higher clock speeds

## Famous Systems
- Mac SE/30 (beloved compact Mac)
- Mac IIci (popular workstation)
- NeXT Computer (Steve Jobs)
- Amiga 3000

## Historical Significance
Defined the "classic" high-end Mac era. The SE/30 and IIci remain legendary among enthusiasts.
